Texsico Refresh Package
=======================

What changed:
- Rebranded the app name from Ripple to Texsico
- Updated auth pages and feed copy to match the new brand
- Switched the UI toward a cleaner, more minimal glassmorphism style
- Removed the heavy ripple/circle visual direction in favor of simpler surfaces
- Added composer draft autosave on the feed
- Added feed smart filters: All, Photos, Discussed, Quick reads
- Added feed density modes: Comfort and Compact
- Added chat conversation search
- Added chat quick reply chips
- Updated manifest and favicon assets for Texsico
- Updated SQL seed branding and database naming to texsico_db

Notes:

- Review config/database.php before deployment and replace credentials if needed.
