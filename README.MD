# Texsico

Texsico is a mini social networking web application developed as an academic project. It allows users to register, log in, create posts, interact through likes and comments, and manage their profile in a simple social platform.

## Live Preview

## **[https://texsico.xyz](https://texsico.xyz)**

## Features

- User registration and login
- Newsfeed posting system
- Likes and comments
- User profile management
- CRUD operations
- Password reset via email

## Built With

- HTML
- CSS
- JavaScript
- PHP
- MySQL
- MVC Architecture

## Purpose

This project was created to apply web development concepts such as authentication, database integration, CRUD functionality, MVC structure, and basic security practices in a working social networking system.
